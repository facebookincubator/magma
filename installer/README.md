# FB-Magma Infra Installer

Installer to configure infra for proper Magma installation

## Getting Started

This installer will set up the underlying infra for the Magma project up and running on your local machine for development and testing purposes.  Please see Important  deployment for notes on how to deploy the project on a production system with automation , HA and scalability..

### Prerequisites

A Baremetal or VM Host (with Intel VT-X) enabled with at least: 
	- 4 CPUs
	- 12GB RAM
	- 80 GB HDD Space

        - 12GB RAM
        - 80 GB HDD Space
